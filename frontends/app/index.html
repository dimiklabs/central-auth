Central Auth Demo

Four independent services, one shared login cookie — the same pattern Google uses for Gmail, Drive, and Docs.

How this works

  1. Click any service below. You'll be redirected to the login form.
  2. Sign in once with alice@example.com / demo123.
  3. The central_auth cookie is set on .centralauth.local.
  4. Click any other service from the nav — no login form, instant access.
  5. Logout from any service clears the cookie everywhere.
